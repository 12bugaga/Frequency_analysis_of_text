Triplet application


********************************************************************************************************
Input parameters: path to the text file.
Output results: display, separated by commas, the 10 most frequent triplets in the text (3 consecutive letters of the word), and on the next line the program runtime in milliseconds.
Requirements: the program must process text in a multithreaded mode, and support the cancellation of processing on pressing any key, after which the current results must be displayed on the screen.
********************************************************************************************************
Входные параметры: путь к текстовому файлу.
Выходные результаты: вывести на экран через запятую 10 самых часто встречающихся в тексте триплетов (3 идущих подряд буквы слова), и на следующей строке время работы программы в миллисекундах.
Требования: программа должна обрабатывать текст в многопоточном режиме, и поддерживать отмену обработки на нажатие любой клавиши, после чего на экран должны выводиться текущие результаты.
